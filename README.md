# Litter Picker

A mobile app for coordinating litter picking activities.

# Goals

I am pursuing the following goals:  

1. I want to learn how to use Claude Code and the agentic software development paradigm 
(Spring 2026)
2. When I have a minimum viable product I want to provide it to the litter picking 
and conservation communities and make it available on the appstore.
3. Initially, I am going to focus on iOS, later Android.
